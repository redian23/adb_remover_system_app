# adb_remover_system_app

[b][size=3][center]Скрипт (Windows) для удаления системных программ. (Miui & Custom Rom's)[/center][/size][/b]
[size=2][center]([color=green]Работает[/color] [color=red]на разблокированном загрузчике[/color])[/center][/size]
[spoiler=Установка]
Просто распаковать архив в любом удобном месте на пк.[/spoiler]
[spoiler=Использование]
1.Заходим в режим разработчика
2. Включаем отладку по Adb
2.1 Проверяем командой [color=blue]adb devices[/color]
3.Подтверждаем сопряжение c компьютером 
4.Запускаем скрипт 
5.Профит [/spoiler]
[spoiler=Примечание]
Можно редактировать  файл .bat в Блокноте.
Можно вписать любую программу зная ее путь. ([color=blue]com.google.app[/color])
Можно закомментировать нужную программу,добавив [color=blue]#[/color] или [color=blue]rem[/color] в начале команды.
[/spoiler]
